type-> exibe o que tem dentro do arquivo

//como usar

type+arquivo

ex: type oi.txt

find/findstr-> retorna a linha que contem a palavra/frase que queremos(se a palavra não existir no arquivo nada será retornado)

// como usar

find+"palavra"+arquivo

ex: find "Hello" oi.txt

ren/rename->usado para renomear um arquivo

//como usar

ren/rename+ arquivo+ novo nome

ex: ren oi.txt h1.txt

ex:prompt dev
